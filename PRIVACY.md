# Privacy Policy — Ainvis Local

**Effective date:** 2026-06-26

Ainvis Local is a local-first Claude Code plugin published by XPAND K.K. This policy explains what data the plugin handles. The short version: **Ainvis Local collects nothing.**

## What we collect

Nothing. Ainvis Local does not collect, transmit, or store any personal data on our servers. We run no server in connection with this plugin, and it contains no telemetry or analytics.

## Where your data lives

Everything you create with Ainvis Local — conversations, meeting logs, minutes, decisions, and any reference files you place in `library/` — is stored as plain Markdown files in your own workspace folder, on your own machine. It never leaves your machine through Ainvis Local. Uninstalling the plugin does not delete this data.

## Third parties

- **Anthropic / Claude.** The executives run inside your own Claude Code, on your own Claude subscription or API key. Your prompts are processed by Anthropic under your own agreement with Anthropic; Ainvis Local is not a party to that traffic and does not receive a copy of it. See Anthropic's privacy policy for how Claude handles your inputs.
- **Fonts (CDN).** The bundled boardroom viewer loads brand fonts from XPAND's content delivery network (`ainvis.ai`). These are static font files; such a request carries only what any ordinary web request carries (for example, your IP address, visible to the CDN) and includes no Ainvis Local content. Offline, the viewer falls back to system fonts.

## The boardroom viewer

The bundled viewer is a read-only local renderer bound to `127.0.0.1` (localhost). It does not connect to Claude/Anthropic and makes no Anthropic API calls. See [SECURITY.md](SECURITY.md).

## Children

Ainvis Local is a developer tool and is not directed to children.

## Changes

We may update this policy. Any changes will be published in this repository.
